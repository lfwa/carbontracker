Getting started

Installation

carbontracker is available on PyPI and can be installed using pip:

pip install carbontracker

To get accurate measurements for applications using GPUs, please make sure NVML is installed.

Example usage (CLI)

See documentation for list of CLI options.

$ carbontracker python script.py

Example usage (Python)

See documentation for CarbonTracker class.

from carbontracker.tracker import CarbonTracker

tracker = CarbonTracker(epochs=max_epochs)
# Training loop.
for epoch in range(max_epochs):
    tracker.epoch_start()
    # Your model training.
    tracker.epoch_end()

# Optional: Add a stop in case of early termination before all monitor_epochs has
# been monitored to ensure that actual consumption is reported.
tracker.stop()
